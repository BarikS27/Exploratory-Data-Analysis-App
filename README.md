# Exploratory-Data-Analysis-App

Description: 
- EDA application that communicates key data insights and visualizations from user-selected files. 

Overview: 
- This Streamlit application allows users to upload a CSV file and instantly explore its numeric data through summary statistics and interactive visualizations. It’s designed to provide quick insights without requiring any coding.

Features: 
- Upload any CSV file containing numeric data
-	Preview the first few rows of the dataset
- Automatically detect numeric columns
- View summary statistics for all numeric variables
- Choose between three visualization types:
- Histogram
- KDE Plot
- Scatter Plot (requires at least two numeric columns)
- Clean, modern UI with centered layout and styled headings
  
How to Run App: 
- Install streamlit on IDE, with "pip install streamlit" in terminal
- Then type "cd" to your local file path in terminal
- Finally run "streamlit run BAN Project - Barik.py" in terminal
